Es un framework de código abierto para la creación de sitios web responsivos y móviles, basado en HTML, CSS y JavaScript.

Git es un sistema de control de versiones que permite rastrear los cambios en archivos, mientras que GitHub es una plataforma web que aloja repositorios Git y facilita la colaboración. En esencia, Git es la herramienta y GitHub es el lugar donde se guarda y trabaja con los repositorios.
